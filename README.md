For saving my analysis options

## Features

analysis_options.yaml

## Getting started

```dart
flutter pub add dev:barqui_analysis_options
```

Then, add include in analysis_options.yaml:

```dart
include: package:barqui_analysis_options/analysis_options.yaml
```
